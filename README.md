# alive
远程虽断，现场犹存 — 赋予无人值守瞬间以永恒

# 快速安装
```
go install github.com/xiaoqidun/alive@latest
```

# 编译安装
```
git clone https://github.com/xiaoqidun/alive.git
cd alive
go build alive.go
```
